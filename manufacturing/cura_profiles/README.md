# Cura Profiles

> **Exploratory profile notes — tune for your specific printer**

The included `.curaprofile` is a starting point for a clearance-test print. Cura profiles can be printer-definition dependent, so import may vary by Cura version and printer setup.

## Suggested starting settings

| Setting | Exploratory starting value |
| --- | ---: |
| layer height | 0.20 mm |
| wall line count | 3 |
| top/bottom thickness | 0.80 mm |
| infill density | 20% |
| infill pattern | grid |
| support | off initially |
| build plate adhesion | skirt or brim, depending on printer |
| material | set in Cura for your filament |
| temperatures | set in Cura for your filament/printer |

## Why support starts off

This is a fit-test part. Supports inside the vial bores could distort the clearance test. If Cura requires support for other geometry, inspect the preview carefully and avoid support material inside the cylindrical bores.

## Profile status

The profile is not validated. Treat it as a starting point only.
