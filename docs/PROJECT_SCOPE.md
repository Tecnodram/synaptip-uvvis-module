# Project Scope

> **Exploratory prototype documentation — derived from Master plan 1**

## Purpose

This project documents the early-stage development workflow for a parametric UV-VIS interoperability adapter concept. The current repository is limited to research organization, reverse-engineering preparation, and traceability infrastructure.

## Source basis

- `Documents/Proyecto.docx` is the original working source document.
- `Master plan 1.txt` is the authoritative operational synthesis for this repository.
- Public files in this scaffold are generalized derivatives and must remain safe for publication.

## Phase 0 — Research & Traceability

Phase 0 establishes:

- repository structure
- document naming and versioning discipline
- prompt traceability
- measurement-capture templates
- public-safe documentation rules

## Phase 1 — Reverse Engineering

Phase 1 establishes:

- generalized geometry reconstruction records
- indexing-system observation templates
- dimensional capture workflow
- first-pass parametric model organization

## Claims policy

The repository must not present:

- final compatibility claims
- validated tolerances
- optical performance conclusions
- manufacturer endorsement
- confidential institutional or research information

Any future performance statement must be backed by documented evidence and versioned review.

## Phase 0 / Phase 1 artifact policy

- Raw source documents and raw photos stay local unless explicitly reviewed for publication.
- Public repository content should use generalized wording, sanitized filenames, and clear validation status labels.
- Dimensional values observed during reverse engineering remain **preliminary** until formally reviewed and logged.
