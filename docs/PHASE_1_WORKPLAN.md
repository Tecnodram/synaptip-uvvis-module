# Phase 1 Workplan

> **Exploratory prototype documentation**

## Objective

Extract the minimum reliable geometry needed to reconstruct the existing carousel architecture and prepare a traceable basis for the first serious parametric model revision.

## Work packages

### WP1 — Datum definition

- define center datum
- define radial datum family
- define provisional angular zero reference

### WP2 — Base reconstruction

- capture outer boundary
- capture concentric ring diameters
- capture total height and local step heights
- capture central fastening geometry

### WP3 — Indexing feature study

- count narrow slots
- locate wider slot
- compare slot pattern against observed tray positions
- classify statements as observation vs hypothesis

### WP4 — Holder envelope definition

- confirm target vial diameter
- confirm required holder height
- confirm optical-window elevation and opening envelope

### WP5 — Parametric model update

- separate base, ring, spoke, slot, and holder modules
- keep all uncertain values explicitly marked as provisional
- do not introduce fit or tolerance claims

## Exit criteria for initial Phase 1 pass

- geometry extraction matrix completed
- unresolved dimensions listed
- starter OpenSCAD file refactored around explicit modules and provisional parameters
- no private photos or raw measurements published
