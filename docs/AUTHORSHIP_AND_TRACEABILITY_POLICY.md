# Authorship and Traceability Policy

> **Exploratory prototype documentation**

## Authorship

The repository preserves design supervision by **Daniel Ramírez Martínez**.

## Traceability requirements

The project must preserve:

- chronological engineering evolution
- prompt traceability
- commit history
- OpenSCAD parametric logic evolution
- validation logs
- engineering rationale

## AI-assisted outputs

AI-assisted outputs must always remain:

- supervised
- reviewed
- curated
- versioned
- documented

AI assistance does not replace human engineering judgment, validation, or final approval.

## Required workflow linkage

Where practical, future work items should link:

- source observations
- prompt log entries
- measurement records
- design decisions
- changed files
- commit hashes
